## [OVERVIEW-010] Introduction

KairosChain is a Meta Ledger for recording AI skill evolution.
It combines a private blockchain (ChronosChain) with a Pure Skills implementation (Ruby DSL/AST).

## [ARCH-010] Architecture

KairosChain consists of:
1. Skills Layer: Ruby DSL definitions of capabilities
2. Blockchain Layer: Immutable record of skill transitions
3. MCP Server: Interface for AI agents
